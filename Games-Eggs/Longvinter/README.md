# Longvinter
[Longvinter] - Fish, farm, craft, gather, cook, loot or steal from other players and build a campsite or a village with your friends in an open-world multiplayer sandbox game without rules!
- Cred to [TuEye](https://github.com/TuEye) as well, Auto Updater script was clutch haha


## Server Port
| Port    | Default |
|---------|---------|
| Game    | 7777    |
| Query   | 27015   |
